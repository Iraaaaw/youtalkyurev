Profile

Salam kenal. Perkenalkan nama saya Ira Wahdani, saya mahasiswi yang mempelajari ilmu bimbingan dan konseling, di Universitas Ibn Khaldun Bogor. Saya tinggal di Bogor, tepatnya di Komplek IPB 1 Loji. Saya sekarang berada di tingkat semester 5. Saya seorang calon konselor profesional.

Saya aktif dalam kegiatan kelas, baik itu sebagai mahasiswa aktif maupun terlibat dalam kegiatan diluar kelas. Saya memiliki beberapa pengalaman diluar kelas, saya ikut berorganisasi, aktif dalam kepanitiaan, terjun sebagai perwakilan untuk menjadi relawan, terjun observasi lapangan dan juga sempat memiliki pengalaman secara langsung menjadi konselor sekolah.

Saya membuat web youtalkyu sebagai tugas UAS dari inovasi BK, juga sebagai sebuah wadah untuk membagikan tips-tips mengenai pribadi, sosial, akademik dan karier. Saya juga membuka layanan konseling ataupun curhat baik melalui chat maupun gmeet.

Terima kasih sudah berkunjung, semoga bermanfaat.
